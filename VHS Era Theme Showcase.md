

A retro, high-contrast dark theme bringing the nostalgic VHS aesthetic to Obsidian.

## Features

The VHS Era theme combines vibrant neon colors with a dark background for optimal readability and style.

### Color Palette

The theme showcases vibrant colors throughout:

- [[Links]] and active tabs use cyan
- ==Highlighted text== uses yellow
- `Code elements` are styled with theme colors
- Task icons like [!] use yellow, [f] uses red, [w] uses green

## Custom Task Icons

The theme includes custom checkbox styling for task management:

- [ ] Uncompleted task
- [x] Completed task
- [!] Important task
- [?] Question or uncertainty
- [*] Star or priority item
- [n] Note to self
- [l] Location marker
- [i] Information
- [S] Savings or money
- [I] Idea or brainstorm
- [p] Pro or advantage
- [c] Con or disadvantage
- [b] Bookmark
- [f] Fire or urgent
- [w] Win or success
- [k] Key point

## Callout Boxes

> [!warning] Warning Callout
> Important warnings stand out with yellow accents to catch your attention.

> [!tip] Tip Callout
> Helpful tips are highlighted in cyan for easy identification.

## Code Blocks

Inline code looks like `this` with proper highlighting.

```javascript
// Code blocks have syntax highlighting
function vhsEra() {
    const theme = {
        name: "VHS Era",
        style: "retro",
        colors: ["cyan", "pink", "purple"]
    };
    return theme;
}
```
